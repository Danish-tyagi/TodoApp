# 📝 Todo App

A simple Todo List web app built using *HTML, **CSS, and **JavaScript*.  
It allows users to add and delete daily tasks quickly.

---

## 🚀 Features

- Add new tasks  
- Delete tasks  
- Clean and responsive UI  
- Real-time task rendering

---

## 💻 Tech Stack

- HTML
- CSS
- JavaScript
